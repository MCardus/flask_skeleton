# Flask API Skeleton

## Flask API Skeleton:

This is a REST API Skeleton implemented in Flask. It has been tested in Python 3.6.4 and Flask 0.12.


Installation manual:

* Install:
   ````
   pip install .
   ````
* Run tests
    ````
   python -m unittest discover
   ````
   
* Run project
    ````
   python run_server.py
   ````

Sample calls:

* foo call:
    ```
    http://localhost:5000/foos?id=11&id=12
    ```
    
* foos call:
    ```
    http://localhost:5000/foos?id=11&id=12
    ```
